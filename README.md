# -# Text Fighting Game (文字格鬥遊戲)

一個基於 Java 基礎語法與物件導向基礎（OOP）開發的命令列（CLI）文字格鬥遊戲。本專案是我在學習 Java 程式設計時的實作練習，旨在透過實戰加強對物件導向特性、流程控制與異常處理的理解。

---

## 🎮 遊戲特色

* **多樣化角色/職業**：支援不同屬性（生命值、攻擊力、防禦力）的角色供玩家選擇。
* **回合制戰鬥系統**：清晰的回合邏輯，玩家可選擇「普通攻擊」、「施放技能」或「防守/補血」。
* **隨機性與暴擊機制**：引入隨機數判定，包含攻擊漂移、暴擊機率以及技能觸發機率，增加遊戲趣味性。
* **互動式終端介面**：使用 `Scanner` 實作流暢的文字選單與戰鬥日誌輸出。

---

## 🛠️ 使用技術與 Java 核心概念

本專案核心運用了以下 Java 程式設計概念：
* **物件導向程式設計 (OOP)**：
    * **封裝 (Encapsulation)**：使用 `private` 屬性並透過 Getter/Setter 存取，確保角色資料安全。
    * **繼承与多型 (Inheritance & Polymorphism)**：(若有使用) 建立基礎 `Hero` 類別，並衍生出不同的職業（如戰士、法師），動態綁定不同的技能傷害計算。
* **流程控制**：利用 `while` 迴圈處理回合循環，`switch-case` 處理使用者選單輸入。
* **隨機數應用**：使用 `java.util.Random` 實作閃避率、暴擊率與傷害浮動。
